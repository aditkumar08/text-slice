## How to Use

1. Click "Choose File" and select any text file (.txt, .log, .csv, .json, .md, etc.)
2. Adjust the percentage slider
3. Choose selection mode
4. Click **Copy to Clipboard** or **Download Portion**

Keyboard shortcuts:
- `Ctrl+C` ‚Üí Copy
- `Ctrl+S` ‚Üí Download
